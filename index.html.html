O que e a calculadora maluc

A calculadora maluca é uma aplicação web simples que realiza cálculos de potenciação, multiplicação, divisão e porcentagem. O código utiliza HTML, CSS e JavaScript para criar a interface e executar os cálculos.

fucionalidade && tecololgia usada

Funcionalidades: Calcula potenciação de dois números. Realiza multiplicação entre dois números. Executa divisão entre dois números. Calcula a porcentagem de um número em relação a outro. Tecnologias Utilizadas O código utiliza as seguintes tecnologias: HTML - Estruturação da página web. CSS - Estilização da página e animações. JavaScript - Lógica de programação para realizar os cálculos.
